Inleiding

Als opdracht moest ik de formulieren website Trees for all coatings testen . Trees for all is een wordPress maatwerk website gemaakt door go2people zelf. Het is website voor de stichting trees for all om te kunnen doneren voor het herstellen van het milieu door bomen te planten. Het was aan mij de taak om de formulieren te gaan testen, en kijken of er geen bugs zijn.

Ga naar wat ik heb geleerd?

Testen

Jira Ticket

Ik heb een Jira ticket gekregen van de projectmanager. Op de Jira heb dan toegang tot het bord. Daar had je verschillende informatie over alles van die ticket.

De opdracht

Voordat ik kon beginnen werd ik kort debrief over de taak. Ik moest de website gaan testen op mobiel en een tablet. Ik had een IPAD en mijn Samsung telefoon. Dus het was ook mogelijk om op verschillende browser te kunnen testen. Daarna moest ik screenshots waar ik bugs kan vinden.

Hoe heb ik getest?

Ik ben begonnen op de tablet. Ik ging de pagina's heen en keek naar de vormgeving, de lettergrootte, uitlijning en de layout. IK heb de dan ook de tablet liggende en staand getest om te kijken wat er gebeurde. Wanneer ik iets raar vindt, maak ik een screenshot en zet een korte beschrijving van de bug. De screenshot heb ik daarna geplaatst op Jira zodat de programmeur het probleem kan aanpakken.

Verder ging ik ook dezelfde doen bij mobiel. Voor mobiel heb ik ook op andere browser getest om te kijken of er niks mis gaat met de vormgeving van de website.

Test bevindingen

Test bevindingen worden op Jira geplaatst. Er waren veel bugs die ik heb gevonden. Maar ik heb niet alles plaatsen want in die begin wist ik niet hoe het ontwerp gaat. Dus ik wilde niet feedback geven op basis van aannames.

Per bevinding heb ik een screenshot gemaakt en daarvan een een korte toelichting van het probleem. Ik beschreef op welke pagina het probleem plaatsvind om meer context te geven aan de programmeurs.

Wat heb ik geleerd?

Met deze taak heb ik geleerd hoe Jira-tickets worden behandeld. Tijdens de opdracht heb ik ook ontdekt hoe je een probleem op een duidelijke manier kunt beschrijven, zodat het begrijpelijk is voor het hele team.
