4stagiairs

Mijn eigen gemaakte website

1. De opdracht

Maak je eigen website op basis van wat je heb geleerd tijdens de stage

Als opdracht kreeg ik de kans om mijn eigen website te maken met alles die ik heb geleerd. Daardoor heb ik verschillende programmeertalen gebruikt onder andere WordPress, SCSS, Javascript, PHP, Javascript, Timber en TWig. Ik kon helemaal los gaan met deze opdracht. Dus ik heb daardoor 4stagiairs gemaakt.

2. Wat is 4stagiairs?

4stagiairs is een informatie website voor alle stagiairs die bij go2people stage gaat lopen. Het doel van deze website is informatie te bieden bijvoorbeeld stageopdrachten, adminatratieve zaken, informatie over de collega's en een overzicht van het stage traject bij go2people.
