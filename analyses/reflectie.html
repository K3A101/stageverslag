Reflectie op lange stage

Competentie: Evalueren

Algemene reflectie

De laatste twintig weken was heel erg leerzaam als frontend developer. De stage was een uitdaging geweest omdat ik wist niet wat ik zou verwachten en het was voor het eerst dat ik mijn programmeurs vaardigheden van Minor Web en Project Tech binnen een organisatie kon gebruiken. Tijdens mij korte stage Mijn korte stage heb ik gewerkt als UX designer en deze keer ben ik overgestapt naar techniek. Ik wist wat ik zou verwachten of ik goed genoeg ben als een frontend development. Maar ik ben verrast in mijn ontwikkeling als een programmeur. Ik heb nieuwe programmeertaal, Javascript Bibliotheken en ook frameworks geleerd waarbij ik nauwkeurig kon toepassen in mijn stageopdrachten en mijn eindproject.

Reflectie op mijn opgedane kennis

Bij project Tech en Minor Web heb ik verschillende programmeertalen en frameworks geleerd zoals Javascript, css, HTML, Nodejs, Express Js, PWa's enzovoorts dus ik heb al een technische achtergrond om een frontend stage te kunnen doen. Tijdens mij stage heb ik nog meer kennis kunnen krijgen. Om aan mij stageopdrachten te kunnen werken, heb ik PHP geleerd om wordPress website te kunnen ontwikkelen, verder met scss gewerkt om een de webpagina's te vormgeven. Scss is een css framework waar je de stylesheets kan verdelen in verschillende bestanden om de code gestructureerd te houden.


Verder heb ik kunnen werken met verschillende javascript bibliotheken zoals javascript en bootstrap. Ik heb geleerd hoe je ze in een project kan implementeren maar ook hoe je ze kunnen gebruiken op basis van de documentatie. Daardoor kon ook mijn stageverslag maken met alle kennis die ik heb opgedaan.


Als laatste heb ik ook samenwerken met collega's van verschillende disciplines (UX designers, ), door vragen te stellen en actief opzoek gaan naar feedback, kon ik de tips en tops in mijn eigen manier van werken gebruiken. Om de cummunicatie tussen de collega's heb ik kunnen werken met verschillende interne programma's zoals Jira, stopwatch, Discord, en Gripp. Ik heb verschillende feedback sessies gehad met developers waar ik mijn code kon beoordelen en ook mijn code kon onderbouwen. Hierbvan heb ik geleerd hoe technische begrippen makkelijk kunnen toelichten aan verschillende betrokken om de boodschap duidelijk naar voren te brengen.


Als laatste heb ik de mogelijkheid gekregen om te werken aan een Wordpress website en die aan het eind te presenteren voor de hele go2people team. Hiermee heb ik een volledige ontwerpproces kunnen uitvoeren en veel iteratie s kunnen met de feedback van mij werk. Ik ben dankbaar voor de kans om deze stage te doen.
